Using Post Submit Actions

  • A Form এর - Data - গুলো সাবমিট করলে কি হবে কোথায় নিয়ে যাবে কিসের সাথে - connect - করবে এসব কিছু - Post Submit Actions - থেকে পড়তে হয়

JetForm a Text Field এর মাধ্যমে কি কি - Type Field ব্যবহার করা যায়

  • Name Field
  • Email Field
  • Number Field
  • Password Field
  • Text Field এর মধ্যে - Advanced Setting - এর মধ্যে Placeholder থাকে

Using Repeater Fields?
